Creative Coding

✻

Creative coding is a program whose purpose is to introduce to girl scouts the many ways computer science can be applied to different devices. We've shown girl scouts how to write and understand code in various programming languages by utilizing Scratch, Raspberry Pi's, Ozobots,Turtle Graphics, and Google Cardboards. Our objective is to get girl scouts more enthusiastic about STEM, especially Computer Science.

✻

Scratch

Scratch is a block based visual programming language that allows girls to create virtual games. This is a great way to be introduced to programming and understanding the basic logic structure.

Raspberry Pi

With Raspberry Pis we introduce bread boards, ultrasonic sensors, Jumper wires, resistors, and led lights .Young girls will use python to program the ultrasonic sensor and the lights on the breadboard. The python script that the girls write will change the LED lights based on the distance of the sensor.
